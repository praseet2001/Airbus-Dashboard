# ✈️ Airbus A220 Automation Tools Dashboard

An interactive and enterprise-ready dashboard built using **Streamlit** to monitor, track, and analyze the lifecycle of automation tools used across Airbus A220 programs.

## 🚀 Overview

This dashboard provides a centralized, real-time view of automation tools across different business functions, governance categories, and lifecycle stages. It enables decision-makers to quickly understand tool maturity, ownership, and completion status.

## 🔧 Key Features

✅ **Live Data Integration**  
- Connects directly to Google Sheets (real-time updates)  
- Automatic fallback to Excel file (offline resilience)

✅ **Interactive Visualizations**  
- Governance-based donut chart  
- Category completion overview (mini donuts)  
- Lifecycle progress tracking with visual indicators  
- Heatmap for tool completion distribution  
- Global map showing tool distribution

✅ **Dynamic Filtering**  
- Filter by Business Function  
- Filter by Governance Category  
- Interactive selection via charts

✅ **Tool Insights Panel**  
- Detailed view of selected tool  
- Lifecycle progress indicator with circular visualization  
- Status-based classification (Operational, Licensed, Evaluation, etc.)

✅ **Enterprise-Ready Design**  
- Clean UI with card-based layout  
- Responsive and interactive  
- Built for executive-level reporting

## 📊 Data Source

The dashboard reads data from:

- ✅ **Google Sheets (Primary Source)**  
- ✅ **Excel File (Backup Source)**
